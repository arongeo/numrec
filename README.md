# numrec
"handwritten" digit recognizer with pygame

Install requirements using ```pip install -r requirements.txt```

Controls:
  - R = resets the board
  - Q = quits the program
  - Left click = Draw
  - Right click = Erase
  
Build Python version is 3.7.6
